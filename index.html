Ce que Hakari
sait de toi

Mis à jour en juin 2025 — Bot Discord d'économie

L'essentiel

Hakari est un bot d'économie et de gameplay. Il stocke des données purement liées au jeu — jamais ton nom, jamais tes messages, jamais ton email. Uniquement ce dont le bot a besoin pour fonctionner.

Données collectées

  • Ton Discord User ID
    Le numéro unique que Discord t'attribue. Permet de lier ta progression à ton compte.
  • L'ID du serveur (Guild ID)
    Pour que chaque serveur ait ses propres données séparées.
  • Économie virtuelle (cash, banque, inventaire)
    Monnaie fictive et objets du jeu. Aucune valeur réelle.
  • Cooldowns de commandes
    Timestamps des dernières utilisations pour gérer les délais entre commandes.
  • Données de gameplay JJK
    Pouvoirs, doigts Sukuna, niveau de sang, stacks — informations de progression dans le jeu.
  • Appartenance entreprise
    Membership fictif dans une entreprise du jeu.
  • Logs d'administration
    Actions effectuées via le panel admin (ID de l'admin, cible, action, date).
Hakari ne stocke jamais ton nom, pseudo, avatar, email, adresse IP, ni aucun message que tu envoies sur Discord.

Pourquoi ces données

Sans ces données, le bot ne peut pas sauvegarder ta progression ni appliquer les cooldowns. C'est le strict minimum pour fonctionner.

Partage & tiers

Aucune donnée n'est partagée, vendue, ni transmise à qui que ce soit. Les données restent sur le serveur d'hébergement du bot, accessibles uniquement aux administrateurs du serveur Discord concerné.

Suppression

Contacte un administrateur de ton serveur Discord pour demander la suppression de tes données. Il peut réinitialiser ton profil complet via le panel d'administration.

Contact

Pour toute question, passe par le serveur de support du bot ou contacte directement l'administrateur de ton serveur.
